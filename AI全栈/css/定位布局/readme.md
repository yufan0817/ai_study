# 定位布局
开发页面的时候，常常需要确定将某一个容器展示在页面的某个地方，如何控制它的位置

1. position: relative; 相对定位：参考自己本身的文档流的位置来定位,原有的文档流位置还存在
2. position: absolute; 绝对定位：找自己的外层拥有定位属性的父容器来做参考，如果找不到，
会以 body 为标准。绝对定位会导致容器脱离文档流
3. position: fixed; 固定定位：以浏览器可视区域为参考标准来定位
4. position: sticky; 粘性定位：以浏览器可视区域为参考标准来定位，
谁在下面谁的优先级更高
内层子容器没法盖住外层父容器
- 层级： z-index
1. 层级要生效，必须要配合定位属性
2. 只能在兄弟元素之间相比较


bg可以颜色和图片，bg-color只能颜色
# 文档流
页面布局默认是从上往下，从左往右的排列，呈现一种流式的排列

1. 块级元素：默认占据父容器的 100% 宽度（一整行），可以设置宽高
2. 行内块级：可以同一行排列，可以设置宽高
3. 行内元素：可以同一行排列，不可以设置宽高
